## Configuring a Custom Domain for GitHub Pages

> The following instructions are adapted from [GitHub Docs](https://docs.github.com/pages/configuring-a-custom-domain-for-your-github-pages-site).

1. In your repository, navigate to the **Settings** page.
2. Under **Pages**, find **Custom domain**.
3. Enter your domain name, then click **Save**.
4. Update your domain's DNS records as described in the documentation.
5. Optionally, enforce HTTPS for security.

For complete and up-to-date instructions, please visit the [official GitHub Pages documentation](https://docs.github.com/pages/configuring-a-custom-domain-for-your-github-pages-site).
